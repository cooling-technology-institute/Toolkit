Menubar

File Menu

New: Opens a new file. The type of file opened depends on the calculation tab that is selected. See Calculation Tabs for file extension types.

Open: Opens a existing file. The type of file opened depends on the calculation tab that is selected.

Save: Save the current data value to the selected file. The type of file opened depends on the calculation tab that is selected.

Save As: Save the current data value to the new selected file. The type of file opened depends on the calculation tab that is selected.

Print: Prints the report for the selected calculation tab.

Print Preview: Displays the report for the selected calculation tab in a print preview dialog.

Exit: Exit the application.

 

Settings Menu

Select between United States Customary Units (IP) and International System of Units (SI). The check indicates the current selection.

Help Menu

Help: Display the Toolkit's help.

About: Information about the current version of the Toolkit.
