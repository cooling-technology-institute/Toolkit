Toolbar

- File New: Opens a new file. The type of file opened depends on the calculation tab that is selected. See Calculation Tabs for file extension types.

- Open File: Opens a existing file. The type of file opened depends on the calculation tab that is selected.

- Save File: Save the current data value to the selected file. The type of file opened depends on the calculation tab that is selected.

- Save File As: Save the current data value to the new selected file. The type of file opened depends on the calculation tab that is selected.

- United States Customary Units (IP): Set the calculation to IP - feet, Fahrenheit

- International System of Units (SI): Set the calculation to SI - meter, Celsius

- Print: Prints the report for the selected calculation tab.

- Calculate Calculate for the selected calculation tab, same as click on the tab's Calculate button.
