Psychrometrics Tab Usage

 

 

There are three options to choose from to arrive at the property data. With all options the user can toggle between barometric pressure and altitude. The first option allows the user to input wet bulb temperature and dry bulb temperature. The second option allows the user to select dry bulb temperature and relative humidity. The third option allows the user to enter saturation enthalpy.
