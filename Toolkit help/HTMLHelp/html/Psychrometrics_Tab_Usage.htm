Psychrometrics Tab Usage

 

 

 

There are three options to choose from to arrive at the property data. With all the calculation options the user can select between barometric pressure and elevation. The first calculation option allows the user to input wet bulb temperature and dry bulb temperature. The second calculation option allows the user to select dry bulb temperature and relative humidity. The third calculation option allows the user to enter saturation enthalpy.
