When first opened the Tower Design Curve Data is blank.

 

 

You must enter a Water Flow Rate to see the form to enter the wet bulb and cold water temperatures.

 

 

The cold water temperature column will only appear when the range value for that column is entered.

 

 

Enter wet bulb temperatures (WBT-1 through WBT-6) for each range. The cold water temperature data must be entered manually in these data boxes if this is the first time use and no input file is available.
