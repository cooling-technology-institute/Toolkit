Performance Test Calculations 2

 

The test data can be used to calculate the test L/G value (L/G(t)). This will be used later for the calculation of tower capability. Using the Performance Curve Tab we can calculate the test L/G easily (this application solves the iterative calculation of the exit wet bulb temperature and exit air properties of an induced draft cooling tower):

 

L/G(t) = L/G(d) x Q(t)/Q(d) x (W(d)/W(t))1/3 x (r(t) /r(d))1/3 x n(t) /n(d)

L/G(t) = 1.37 x 8,950/10,000 x (136/115)1/3 x (0.06961/0.06922)1/3 x (14.9298/15.0625)
