Introduction to CTI Toolkit 4.0

 

Upon opening, the Toolkit application presents the user with four tabbed applications:

 

Page Layout Overview - an overview of the section of the toolkit program.

Psychrometrics – a versatile air properties calculator

Merkel – a KaV/L calculator

Demand Curve – a graphical worksheet that facilitates cooling tower calculations

Mechanical Draft Performance Curve – an automated performance curve test method analyzer

Converting Data File - The data file format has change. Instructions on how to convert your files to the new format.
