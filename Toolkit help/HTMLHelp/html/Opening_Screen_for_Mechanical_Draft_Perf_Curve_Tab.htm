Opening Screen for Mechanical Draft Performance Curve Tab

 

This application has two data entry screens that are used for the entering of the design, test & curve data. The opening screen (the one that comes into view when the performance curve tab is selected) is the main worksheet. Click on the various data and control areas of the screenshot below to see help text on each.

 

The best way to familiarize yourself with this application is to load one of the examples that came with this program. These examples correspond to the examples in the ATC-105 Appendix C for SI data and Appendix D for IP (inch-pound) data. To access saved data file (*.mdpc), click on the Open File icon on the toolbar or File, Open.. on the menubar. Valid Design Data must be entered for the Calculate, View Graphs..., and the adding test points are activated.

 

For each test point the following are calculated:

  • Adjusted Flow

  • Predicted Flow

  • Tower Capabilities.

  • Cold Water Temperature Deviation. This value represents the cold water temperature that would be expected if the cooling tower were operated at its exact design conditions in its tested physical condition.
